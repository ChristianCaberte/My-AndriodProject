About the Game

Bounce Your Balls is an exciting Android game developed as a project for the Mobile App Development course. It challenges players to control the ball as it speeds up, ensuring it doesn't fall, with unique gameplay mechanics and an immersive experience.

Features

  • Innovative gameplay mechanics
  • Bounce the ball and control it
  • Beautiful graphics and animations
  • Engaging sound effects and music

Development Tools

  • Android Studio
  • C++
  • Unity Hub (2D Game)
  • Visily
